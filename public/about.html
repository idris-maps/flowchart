About

Flowchart is a tool to easily draw relations between nodes. The resulting SVG is unopinionated enough to integrate seamlessly in your design.

The minimal features are intended. If you are looking for something fancy, checkout graphviz or mermaid.

Syntax

Every line represents a relation between two nodes as: 

FROM LINK TO

where FROM and TO can be defined either as: ID[LABEL] or just as: LABEL.
Labels only need to be defined only once.

This:

A[Alice] -> B[Bob]
B -> A

is equivalent to:

Alice -> Bob
Bob -> Alice

Both will result in:

LINK
s can be a line, dashed or dotted and with or without arrow.

Alice -> Bob
            
Alice - Bob
            
Alice --> Bob
            
Alice -- Bob
            
Alice ..> Bob
            
Alice .. Bob

The only configuration is dir, the direction of the graph. It defaults to TB (top to bottom).

dir: TB
Alice -> Bob
            
dir: BT
Alice -> Bob
            
dir: LR
Alice -> Bob
            
dir: RL
Alice -> Bob

Styling

The resulting SVG has no opinion on style. When integrated in a web page, colors and font depend on the parent CSS.
